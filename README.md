# Premium Aluminium Glazing Website 

A **modern, minimal, and luxury-focused** European aluminium glazing website showcasing high-end architectural glass solutions. Developed the **frontend** using modern web technologies for a sleek, responsive, and interactive user experience.

---

## Features 

* Modern and minimal luxury design
* Fully responsive across devices
* Interactive UI elements and animations
* Smooth navigation and scrolling
* Showcases high-end architectural glass solutions

---

## Tech Stack 

* **Frontend:** React, Tailwind CSS, Framer Motion
* **UI Components:** ShadCN UI
* **Animations:** Framer Motion
